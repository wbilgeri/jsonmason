🧱 JSON Mason — Help
📂 Loading a File
Drop any .json file onto the drop zone, or click it to browse. Once a file is loaded, use the Open file button in the toolbar to load a different one. Large files show a progress bar while parsing.
🌲 Navigating the Tree
Click the ▶ triangle next to any object or array to expand or collapse it. Use the toolbar depth buttons — Depth 1 / 2 / 3 — to show only the top levels, or Expand all / Collapse all to toggle everything at once. For very large trees a warning will appear before rendering to avoid freezing the browser.
🔍 Search
Type in the Search box to highlight all nodes whose key or value contains your query. Press Enter to jump to the next match, Shift + Enter for the previous match. The match counter shows current / total.
✏️ Editing Values & Keys
Double-click any value to edit it inline. An input field appears — press Enter to confirm or Escape to cancel. Double-click a key name to rename it. Boolean values can be toggled with a single click. The 💾 Save JSON * button lights up whenever there are unsaved changes.
➕ Adding & Removing Nodes
Hover over any object or array row to reveal the + button on the right — click it to add a new child. You can also right-click any node and choose Add child node. To delete a node, right-click it and choose Delete node (shown in red).
📋 Right-Click Context Menu
Right-click any node to access the full action menu:
  • Save branch as JSON — download just this subtree as a .json file.
  • Save array as JSONL — available on array nodes; saves each item as a newline-delimited JSON record.
  • Copy path — copies the dot-notation path (e.g. data.users[0].name) to the clipboard.
  • Copy value / Copy key — copies the raw value or key string.
  • Rename key / Edit value — same as double-clicking.
  • Copy branch / Paste into node — internal clipboard for moving subtrees around.
  • Duplicate node — inserts a copy of the node next to the original.
  • Expand / Collapse subtree — expands or collapses the entire subtree under this node.
💾 Saving Your Work
Click 💾 Save JSON (or 💾 Save JSON * when there are unsaved edits). A dialog lets you choose the filename and format: Pretty (2-space indented, human-readable) or Compact (minified, no whitespace).
